En redes y tecnología de la información existen **muchos términos y protocolos**. No es necesario aprenderlos todos de golpe, pero conocer los esenciales permite comprender cómo funcionan los sistemas de comunicación modernos. A continuación, se presenta una guía organizada de los protocolos y términos más importantes.

> ⚠️ Nota: Esta lista no es exhaustiva. A medida que avances, se recomienda ir ampliando el conocimiento con protocolos adicionales.

---

## 1️⃣ Protocolos de seguridad y autenticación

|Protocolo|Acrónimo|Explicación|
|---|---|---|
|**Wired Equivalent Privacy**|WEP|Protocolo antiguo de seguridad inalámbrica, protege redes Wi-Fi mediante cifrado. Hoy en día se considera inseguro.|
|**Wi-Fi Protected Access**|WPA|Protocolo de seguridad Wi-Fi más moderno que WEP, usa contraseñas y cifrado más fuerte.|
|**Temporal Key Integrity Protocol**|TKIP|Protocolo de cifrado usado en redes inalámbricas, menos seguro que WPA2/WPA3.|
|**Extensible Authentication Protocol**|EAP|Framework para autenticación, soporta múltiples métodos: contraseñas, certificados digitales, OTP, etc.|
|**Lightweight Extensible Authentication Protocol**|LEAP|Variante propietaria de EAP creada por Cisco para redes inalámbricas.|
|**Protected Extensible Authentication Protocol**|PEAP|Crea un túnel cifrado seguro para autenticación en redes Wi-Fi.|
|**Terminal Access Controller Access-Control System**|TACACS|Protocolo de autenticación centralizada, controla acceso y registra actividades en red.|
|**Internet Protocol Security**|IPsec|Protocolo para comunicaciones cifradas a nivel IP, usado en VPNs para crear túneles seguros.|
|**Internet Key Exchange**|IKE|Protocolo que establece la conexión segura entre dos equipos en VPN, maneja autenticación y cifrado.|

---

## 2️⃣ Protocolos de transferencia y comunicación de datos

|Protocolo|Acrónimo|Explicación|
|---|---|---|
|**File Transfer Protocol**|FTP|Transferencia de archivos entre sistemas.|
|**Simple Mail Transfer Protocol**|SMTP|Envío y recepción de correos electrónicos.|
|**Hypertext Transfer Protocol**|HTTP|Protocolo cliente-servidor para transmitir información en la web.|
|**Server Message Block**|SMB|Comparte archivos, impresoras y recursos en una red local.|
|**Network File System**|NFS|Permite acceder a archivos remotos como si estuvieran en la máquina local.|
|**Network News Transfer Protocol**|NNTP|Distribución y lectura de mensajes en grupos de noticias de Internet.|
|**Remote Shell**|RSH|Permite ejecutar comandos de manera remota en sistemas Unix.|
|**Secure Shell**|SSH|Alternativa segura a RSH, cifrado para administración remota y transferencia de datos.|
|**Asynchronous JavaScript and XML**|AJAX|Técnica de desarrollo web para crear páginas dinámicas usando JS y XML/JSON.|
|**Internet Server Application Programming Interface**|ISAPI|Permite crear extensiones web de alto rendimiento en servidores.|

---

## 3️⃣ Protocolos de enrutamiento y redes

|Protocolo|Acrónimo|Explicación|
|---|---|---|
|**Routing Information Protocol**|RIP|Protocolo de enrutamiento de vector-distancia usado en LAN y WAN.|
|**Open Shortest Path First**|OSPF|Protocolo IGP para enrutar tráfico dentro de un Sistema Autónomo (AS).|
|**Interior Gateway Routing Protocol**|IGRP|Protocolo propietario de Cisco para enrutar dentro de un AS.|
|**Enhanced Interior Gateway Routing Protocol**|EIGRP|Protocolo avanzado de Cisco, mezcla vector-distancia y estado de enlace.|
|**Spanning Tree Protocol**|STP|Garantiza topología sin bucles en redes Ethernet Layer 2.|
|**Cisco Discovery Protocol**|CDP|Permite a administradores descubrir y gestionar dispositivos Cisco en la red.|
|**Hot Standby Router Protocol**|HSRP|Protocolo de redundancia de routers Cisco, asegura disponibilidad de gateway.|
|**Virtual Router Redundancy Protocol**|VRRP|Asignación automática de routers disponibles para hosts, alternativa a HSRP.|

---

## 4️⃣ Redes virtuales y segmentación

|Protocolo/Tecnología|Acrónimo|Explicación|
|---|---|---|
|**Virtual Local Area Network**|VLAN|Segmenta una red física en múltiples redes lógicas.|
|**VLAN Trunking Protocol**|VTP|Mantiene consistencia de VLANs en switches múltiples.|
|**Virtual Private Network**|VPN|Permite conexión segura y cifrada a otra red a través de Internet.|
|**Point-to-Point Tunneling Protocol**|PPTP|Protocolo de túnel seguro usado en VPNs.|
|**Network Address Translation**|NAT|Permite que múltiples dispositivos compartan una sola IP pública para Internet.|

---

## 5️⃣ Gestión de redes y tiempo

|Protocolo/Tecnología|Acrónimo|Explicación|
|---|---|---|
|**Simple Network Management Protocol**|SNMP|Gestiona dispositivos de red (switches, routers, servidores).|
|**Network Time Protocol**|NTP|Sincroniza la hora de computadoras en una red.|
|**Systems Management Server**|SMS|Herramienta para gestionar sistemas, redes y dispositivos móviles.|
|**Microsoft Baseline Security Analyzer**|MBSA|Detecta vulnerabilidades de seguridad en Windows y redes.|

---

## 6️⃣ Control industrial y telecomunicaciones

|Protocolo/Tecnología|Acrónimo|Explicación|
|---|---|---|
|**Supervisory Control and Data Acquisition**|SCADA|Sistemas para monitorizar y controlar procesos industriales.|
|**Session Initiation Protocol**|SIP|Señalización para llamadas de voz, video y multimedia sobre IP.|
|**Voice Over IP**|VOIP|Tecnología que permite llamadas telefónicas a través de Internet.|

---

## 7️⃣ Identificación de recursos y web

|Protocolo/Tecnología|Acrónimo|Explicación|
|---|---|---|
|**Uniform Resource Identifier**|URI|Sintaxis para identificar recursos en Internet.|
|**Uniform Resource Locator**|URL|Subconjunto de URI que identifica páginas o recursos, incluye protocolo y dominio.|
|**Carriage Return Line Feed**|CRLF|Secuencia de control para indicar fin de línea en ciertos archivos de texto.|

---

## 8️⃣ Seguridad, cifrado y túneles

|Protocolo/Tecnología|Acrónimo|Explicación|
|---|---|---|
|**Pretty Good Privacy**|PGP|Programa de cifrado para correos, archivos y datos.|
|**Generic Routing Encapsulation**|GRE|Encapsula datos transmitidos dentro de túneles VPN.|
